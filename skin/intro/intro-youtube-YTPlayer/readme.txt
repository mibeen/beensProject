■ 설치 및 적용


/skin/intro 폴더 에 폴더 채 올리고, 관리자 > 테마관리 > 기본설정에서 인트로로 지정해 주시면 됩니다.


■ 스킨 수정


1. 이미지, CSS, JS 등과 관련된 주소는 상대주소를 입력해도 자동으로 인식합니다. ex) <img src="./img/test.jpg"> → 인트로 스킨 내 /img 폴더 안에 있는 test.jpg 이미지

2. 유튜브 동영상은 아이디만 넣어 주셔도 되며, 아이디 순서에 시작위치 순서를 같도록 맞추어 주셔야 합니다.

3. 텍스트 등은 intro.html 파일에서 직접 수정하셔야 합니다.

4. 사이트 입장 주소는 <a href="<?php echo $enter_url;?>"> 로 걸어 주시면, 커뮤니티와 쇼핑몰 주소를 자동으로 인식합니다.

5. html 문서이기 때문에 PC에서 브라우저로 intro.html 파일을 열어서 수정작업을 하셔도 되나, 동영상은 웹일 때만 실행될 것입니다.
